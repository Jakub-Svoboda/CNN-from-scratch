SFC project - CNN from scratch
Author: Jakub Svoboda - xsvobo0z

Installation:
	In the project folder, make dataset folder:
		mkdir dataset
	Download dataset from  https://www.fit.vutbr.cz/study/courses/SFC/private/benchmarks/mnist/mnist.zip into dataset folder
	From dataset folder: 
		unzip mnist.zip
	Run the project from the main folder:
		python3 main.py
		python3 main.py --train 10
